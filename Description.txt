Программа направлена на помощь преподавателю в проведении тестов, направленных на проверку усвоения материала лекций студентами 1-2 курсов.

Программа включает в себя 2 интерфейса: интерфейс преподавателя, интерфейс студента.

ИНТЕРФЕЙС ПРЕПОДАВАТЕЛЯ-----------------------------------------------------------------------------------------------------------
Программа должна получать от преподавателя:
1)тему, к которой относится вопрос
2)текст задания с местами для подставления данных, генерирующихся программой
3)функцию для генерации подставляемых данных
4)функцию для решения задания программой и дальнейшего сравнения ответа студента с правильным

Вопрос1: как правильней указать места, в которые подставляем данные?
Вопрос2: нужна ли возможность указания нескольких мест для генерации? но судя по вопросам, нужна.
Вопрос3: нужно ли проверять валидность пвведенных функций

Пример вопросов:
	Значение величины a равно 5. Найдите значение величины b после исполнения серии команд:
	if a>5 then b:=a-5 else b:=a*2;

	Пусть a='гардемарин'. Введите значение переменной a после исполнения следующей команды:
	a:=copy(a,1,4)+copy(a,length(a)-1,2)+copy(a,2,1);

	При нахождении наилучшей приближающей линейной функции для заданной таблицы найти коэфицент Мy
	X -3 -1  1  3  4
	Y  1  2  3  4  5

	При нахождении функции вида y=ax+b найти коэфицент b, если Mxx=3, Mxy=5, My=3, Mx=-1.

ИНТЕРФЕЙС СТУДЕНТА------------------------------------------------------------------------------------------------------------------
При проведении тестирования в студенческом интерфейсе программа должна:
1)Запросить данные о студенте (ФИ)
2)Предложить студенту выбрать тему тестирования
3)По порядку выдать все вопросы и принять все ответы студента
4)Выставить оценку

При организации одного вопроса программа:
1)Определяет, какой вопрос давать
2)с помощью процедуры по генерации данных сгенерировать данные для вопроса
3)с помощью процедуры для решения получить правильный ответ
4)показать студенту вопрос с заполненными (уже сгенерированными) данными
5)получить ответ студента
6)определить правильность ответа студента
